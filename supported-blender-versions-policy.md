# Blender Add-on Supported Blender Versions Policy
_For Hubs Foundation (Volunteer) Staff_

## Supported versions

All the versions from a team decided starting version to the latest version of Blender.

The exact versions that are supported are indicated in the readme of the repository for the Blender add-on.

## Unsupported versions

Versions that are no longer supported are removed from regular testing/development and are basically ignored, but aren't intentionally prevented from working.
